# 📊 Fundamentals of Data Analytics Assignment 

## 📌 Overview
This project focuses on applying statistical techniques using Python to analyze business datasets. The goal is to derive meaningful insights and support data-driven decision-making.

The analysis covers descriptive statistics, hypothesis testing, correlation, outlier detection, and visualization.

---

## 🎯 Objective
- Apply statistical concepts to real-world datasets
- Perform hypothesis testing and interpret results
- Analyze relationships between variables
- Provide business recommendations based on data

---

## 📁 Datasets Used

| Dataset Name | Description |
|-------------|------------|
| sales_data.csv | Customer purchase details |
| employee_performance.csv | Employee training performance |
| survey_data.csv | Customer preferences survey |
| regional_sales.csv | Sales across different regions |
| student_behavior.csv | Study hours vs marks |

---

## 🧪 Analysis Performed

### 🔹 Part 1: Descriptive Statistics
- Mean, Median, Variance, Standard Deviation
- Skewness analysis (Mean vs Median)
- Variability comparison
- Business interpretation of dispersion

---

### 🔹 Part 2: Hypothesis Testing (Paired t-test)
- Compared Before vs After training scores
- Hypotheses:
  - H₀: Training has no effect
  - H₁: Training has an effect
- Calculated:
  - t-statistic
  - p-value
- Decision based on α = 0.05

---

### 🔹 Part 3: Chi-Square Test
- Tested relationship between Gender and Product Preference
- Created contingency table
- Calculated expected frequencies and chi-square statistic
- Checked independence of variables

---

### 🔹 Part 4: ANOVA
- Compared sales across regions
- Hypotheses:
  - H₀: All regions have equal sales
  - H₁: At least one region differs
- Computed F-statistic and p-value

---

### 🔹 Part 5: Correlation Analysis
- Pearson Correlation (linear relationship)
- Spearman Correlation (monotonic relationship)
- Strength and direction of relationships analyzed

---

### 🔹 Part 6: Outlier Analysis
- Identified outliers using statistical methods
- Studied impact on:
  - Mean
  - Standard deviation
- Compared results before and after removing outliers

---

### 🔹 Part 7: Statistical Decision Making
Key business questions answered:
- Should training programs continue?
- Which region needs improvement?
- Is product preference dependent on gender?

---

### 🔹 Part 8: Data Visualization
- Histogram: Order Amount distribution
- Box Plot: Regional sales comparison
- Scatter Plot: Study hours vs marks
- Heatmap: Correlation between variables

---

## 📊 Tools & Technologies
- Python 🐍
- Pandas
- NumPy
- SciPy
- Matplotlib / Seaborn

---

## 📈 Key Insights
- Training effectiveness evaluated using t-test
- Regional performance differences identified via ANOVA
- Customer behavior analyzed using chi-square test
- Strong/weak correlations detected between variables
- Outliers significantly impact statistical measures

This project demonstrates how statistical analysis can transform raw data into actionable business insights.
